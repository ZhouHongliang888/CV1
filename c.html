Detect network security system

Values

Ensuring information security can create a lot of value for enterprises and reduce the losses of many enterprises. The industrial chain is generated indirectly through the platform, from which fees can be charged. Actively promote the classified protection of information security.

Methods

Continuously improve their understanding of network information security management through thinking flow chart.

Specific software and hardware：

NMap 1: Able to identify hosts on the network 2: Discover network mapping and inventory, maintain and manage assets 3: Generate analysis of host traffic, response time and other indicators in the network 4: Identify the open port on the target host 5: Search and exploit vulnerabilities and risks in the network
Wireshark 1: Provide rich VoIP analysis 2: Provide real-time capture and offline inspection 3: Ability to examine hundreds of protocols in depth 4: It can run on a variety of operating systems and their versions
Metasploit 1: For windows, Linux and Mac OS X, there are many options 2: Provide modular browser 3: Support basic mining and manual mining
Netsparker 1: Provides advanced web service scanning, vulnerability assessment, HTTP request generator 2: Fully support HTML5 and be able to integrate the software development life cycle 3: Provide manual test and report
